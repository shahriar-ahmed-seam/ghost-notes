# Store screenshots

Drop the final PNGs here using the exact filenames below, then they'll render
in the project README gallery and can be uploaded to the Chrome Web Store.

## Requirements
- Format: **PNG** (or 24-bit JPEG)
- Size: **1280×800** (preferred) or **640×400** — the store rejects other sizes
- At least 1, up to 5 screenshots

## Recommended set (what you already captured)

| Filename | What it shows |
| --- | --- |
| `01-note-on-page.png` | A colour note pinned to a real page with the scope dropdown + live word count |
| `02-popup-dark.png` | The manager popup (dark theme) with stats, search, and scope filter |
| `03-note-colors.png` | A note with the 7-colour palette open |
| `04-popup-with-notes.png` | Popup listing saved notes grouped by site |
| `05-settings.png` | The settings page (appearance, behaviour, shortcuts, backup) |

Tip: capture at 1280×800 by setting the browser window to that size, or take a
full screenshot and crop/scale to 1280×800.
